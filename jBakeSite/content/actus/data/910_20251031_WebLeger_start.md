Après plusieurs années de pauses, le projet "outils de création de site web éco-conçu" reprend du service.

L'objectif est toujours le même créé un outil (pour les développeurs et idéalement pour les utilisateurs finaux) permettant de créer des sites web éco-conçu en utilisant une technique de "génération statique du code".

Dans un premier temps, le but est de créer les "template" et composants utiles pour qu'une "agence web" puisse utiliser l'outil pour créer des sites web pour de petites structures (association, auto-entrepreneur, petit commerçant et artisan, site institutionnel).

WebLeger est l'outil de base pour l'agence web EthiNet.
